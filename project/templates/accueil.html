Bienvenue sur le site Pixoglyphe

Le site Pixoglyphe vous permet d'analyser hiéroglyphes et scènes complètes ! Pour cela, il vous suffit de vous rendre dans les onglets Analyse et Détection. L'onglet Analyse permet d'identifier les différents contextes d'utilisation d'un hiéroglyphe. L'ensemble des signes connus par le modèle de classification est disponible dans la bibliothèque des signes. La partie détection, quant à elle, vous permettra d'analyser une scène complète et d'identifier les différents hiéroglyphes qu'elle contient. L'historique de vos analyses de similarité vous est proposé dans l'onglet Historique où vous retrouverez l'image que vous avez analysée ainsi que la classe prédite par le modèle.

N'oubliez pas, vous pouvez nous aider activement en vous rendant dans l'onglet Labellisation !
