A propos

Ce site web a été créé dans le cadre d'un travail universitaire de Master 1 MIASHS par Fleur Brun, Pricille Franc, Gaëlle Novales, Théo Oriol et Sarah Ramirez, sous la tutelle de Sébastien Biston-Moulin et Maximilien Servajean.

Ce travail s'inscrit dans le projet Karnak, dirigé par Sébastien Biston-Moulin et mené en coopération entre le CNRS, l’Université Paul Valéry et le Ministère de la Culture et de la Communication. Le programme Karnak consiste à recenser et éditer toutes les inscriptions présentes sur les différents monuments du temple de Karnak. Cette démarche a pour but d’aider les égyptologues dans leur travail de recherche et leur permet de disposer d’une base de données de tous les monuments et signes présents dans le temple sous la forme d’une interface web.

C’est dans ce contexte que ce site web a été réalisé, comme outil complémentaire proposant deux principales fonctionnalités. La première est de fournir aux experts la possibilité de visualiser un hiéroglyphe dans ses différents contextes d'apparition, en identifiant à partir d’une image de hiéroglyphe des images similaires. La deuxième fonctionnalité permet de détecter (et reconnaitre) les différents hiéroglyphes présents sur une plaque entière.

Source des données : Les images récoltées pour mettre en place ces outils proviennent du temple de Karnak via le site web dédié.
